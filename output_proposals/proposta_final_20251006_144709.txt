1. OBJETIVO DO PROJETO
----------------------------------------------------------------------

O objetivo do projeto é desenvolver uma API RESTful centralizada que permita a uma aplicação cliente autorizada gerenciar de forma segura o fluxo de recuperação de senha para usuários que existem em múltiplas instâncias de sistema com bancos de dados independentes. A solução garantirá a segurança e a integridade dos dados dos usuários, utilizando tokens de segurança temporários para autenticação sem trafegar senhas pela rede.

----------------------------------------------------------------------
2. FUNCIONALIDADES DA SOLUÇÃO
----------------------------------------------------------------------

**Módulo 1: Solicitação de Recuperação de Senha**
- Iniciar fluxo de recuperação de senha: Permitir que a aplicação cliente envie identificadores de usuário para iniciar o processo de recuperação.
- Geração de Token Seguro: Gerar um token criptograficamente seguro e enviar um link de redefinição para o e-mail do usuário.

**Módulo 2: Redefinição de Senha**
- Validação de Token: Validar o token recebido pelo usuário para garantir sua autenticidade e validade.
- Atualização de Senha: Permitir que o usuário defina uma nova senha, que será atualizada em todas as instâncias do sistema.
