======================================================================
                  PROPOSTA TÉCNICA E COMERCIAL

CLIENTE: Bridgestone
PROJETO: Sistema de Coleta e Relatórios de Dados de Produção
DATA: 02 de Outubro de 2025
VALIDADE: 15 dias

----------------------------------------------------------------------
1. OBJETIVO
----------------------------------------------------------------------

Desenvolver um sistema que coleta dados de 5 máquinas a cada 5 segundos, armazena essas informações no SQL Server e permite a visualização dos resultados em relatórios tabulares.

----------------------------------------------------------------------
2. FUNCIONALIDADES POR MÓDULO
----------------------------------------------------------------------

A solução será entregue nos seguintes módulos funcionais:

**Módulo 1: Coleta de Dados**
- **Funcionalidade 1.1:** Comunicação automática com as máquinas de produção via protocolos industriais.
- **Funcionalidade 1.2:** Captura de dados de até 300 tags a cada 5 segundos.

**Módulo 2: Armazenamento de Dados**
- **Funcionalidade 2.1:** Modelagem de banco de dados no SQL Server 2019.
- **Funcionalidade 2.2:** Implementação de índices e rotinas de limpeza periódica.

**Módulo 3: Relatórios**
- **Funcionalidade 3.1:** Desenvolvimento de até 12 relatórios em SSRS.
- **Funcionalidade 3.2:** Visualização de dados em formato tabular.

**Módulo 4: Testes e Suporte**
- **Funcionalidade 4.1:** Condução de testes integrados in loco.
- **Funcionalidade 4.2:** Suporte ao Go Live e treinamento de usuários.

----------------------------------------------------------------------

----------------------------------------------------------------------
