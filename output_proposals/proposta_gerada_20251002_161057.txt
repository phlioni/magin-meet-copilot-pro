                  PROPOSTA TÉCNICA E COMERCIAL

CLIENTE: Bridgestone
PROJETO: Sistema de Coleta e Análise de Dados de Produção
DATA: 02 de Outubro de 2025
VALIDADE: 15 dias

----------------------------------------------------------------------
1. OBJETIVO
----------------------------------------------------------------------

Desenvolver um sistema para a Bridgestone que permita a coleta automática de dados de cinco máquinas de produção a cada 5 segundos, armazenando essas informações em um banco de dados SQL Server 2019 otimizado. O sistema também deve possibilitar a visualização dos resultados em até 12 relatórios tabulares desenvolvidos em SSRS, garantindo a integridade, consistência e segurança dos dados.

----------------------------------------------------------------------
2. FUNCIONALIDADES POR MÓDULO
----------------------------------------------------------------------

A solução será entregue nos seguintes módulos funcionais:

**Módulo 1: Coleta de Dados**
- **Funcionalidade 1.1:** Integração com CLPs Rockwell para captura automática de dados de produção via FTTM/RSLINX ou OPC Router/Kepware.
- **Funcionalidade 1.2:** Monitoramento e tratamento de falhas na coleta de dados, com geração de logs.

**Módulo 2: Armazenamento de Dados**
- **Funcionalidade 2.1:** Modelagem e otimização do banco de dados SQL Server 2019, incluindo índices e rotinas de limpeza.
- **Funcionalidade 2.2:** Garantia de integridade e segurança das informações armazenadas.

**Módulo 3: Relatórios e Visualização**
- **Funcionalidade 3.1:** Desenvolvimento de até 12 relatórios tabulares em SSRS para consumo dos usuários finais.
- **Funcionalidade 3.2:** Interface de usuário para visualização e interpretação dos dados coletados.

**Módulo 4: Treinamento e Suporte**
- **Funcionalidade 4.1:** Treinamento dos usuários para uso da interface e interpretação dos dados.
- **Funcionalidade 4.2:** Suporte ao Go Live e assistência inicial aos usuários.
